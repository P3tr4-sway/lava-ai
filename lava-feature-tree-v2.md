## 一、首次进入 / 启动

### 首页加载状态
- 有推荐数据 → 展示「为你推荐」卡片列表
- 无推荐数据 → 展示空态占位，引导搜索
- loading 中 → 卡片骨架屏，不显示操作按钮

### 搜索入口
- 搜索框占位文案：「What do you want to play?」
- 快速标签（Wonderwall / Blues in E / …）
  - 点击标签 → 直接填入搜索框并提交 → 跳转搜索结果页 `/search?q=…`
- 手动输入 + 回车/点击发送 → 同上跳转

### 继续播放（有历史记录时显示）
- 展示最近一首歌的封面、标题、进度条
- 无历史记录 → 该区域不展示
- 点击 → 恢复播放，跳转 `/play/:id`，进度条从上次位置恢复

### 为你推荐
- 展示 4 张歌曲卡片
- 每张卡片元素：封面渐变色、歌名、艺术家、难度星级、调式、风格标签
- 点击卡片 → 跳转 `/play/:id`（加载静态曲谱）
- 数据不足 4 张 → 有几张展示几张，不补占位卡

## 二、搜索结果页 `/search`

### 搜索栏（顶部常驻）
- 带返回按钮 + 文字输入框
- 进入页面时自动聚焦键盘
- 修改内容 + 提交 → 刷新结果列表

### 区域 1：顶部结果卡（最佳匹配）
- YouTube 缩略图（无图时渐变色兜底）
- 右下角显示时长角标
- 悬浮/点击封面 → 展示播放按钮 + 动态波形动画（预览状态）
- 「View Sheet Music」按钮
- YouTube 外链图标 → 打开系统浏览器

### 区域 2：结果列表（3 列网格）
- 每项：缩略图 / 标题（2 行截断）/ 频道名 / 播放量 / 上传时间
- 悬浮 → 出现播放覆盖层图标
- 列表为空 → 展示「无结果」空态，提示更换关键词

### 区域 3：歌曲操作弹窗（选中一首后弹出）

#### 选项 A：AI 乐谱 + 伴奏轨（主推，高亮展示）
- 说明文案：「自动检测调式、BPM，生成和弦谱 + 匹配伴奏」
- 点击确认 → 弹窗内进入进度状态
- 进度步骤动画（依次点亮）：
  - 步骤 1：下载中
  - 步骤 2：分析和弦
  - 步骤 3：检测节拍
  - 步骤 4：处理中
  - 步骤 5：完成
- 每步有时间估算文案（「约 30 秒」）
- 中途失败 → 展示错误信息 + 「重试」按钮
- 完成 → 自动跳转 `/play/:id?generate=1`，弹窗关闭

#### 选项 B：仅生成伴奏轨
- 跳过乐谱生成，仅处理音频
- 步骤较少（下载中 → 处理中 → 完成）
- 完成 → 跳转 `/play/:id`

#### 关闭弹窗
- 点击遮罩或关闭按钮 → 弹窗关闭，返回结果列表，不触发任何生成

## 三、播放 / 学习页 `/play/:id`

### 区域 1：顶栏

#### 元数据栏
- 展示：歌名、艺术家名
- AI 分析来源 → 额外显示「AI Generated」标签
- 调式 / 拍号 / BPM → 可点击进入编辑状态（下拉选择）
- 编辑后 → 更新 DAW 面板的 BPM，节拍网格同步重算

#### 操作区
- 保存按钮
  - 有未保存更改 → 按钮旁显示橙色圆点
  - 点击 → 保存至「我的项目」，圆点消失
- 视图切换：Lead Sheet ↔ PDF 乐谱
  - 无 PDF → PDF 选项置灰，悬浮提示「上传乐谱后可用」

### 区域 2：和弦网格（Lead Sheet 视图）

#### 段落标题色
- 前奏 → 蓝色
- 主歌 → 紫色
- 副歌 → 橙色
- 桥段 → 绿色
- 尾奏 → 粉色
- 自定义 → 灰色

#### 小节格
- 展示该小节绑定的和弦（如 `Am7 Dm`）
- 空小节 → 展示占位虚线框
- 点击小节 → 音频跳转至该小节对应时间点（需有音频轨）
- 无音频轨时点击小节 → 无跳转，仅高亮选中

#### 加载状态（来自 AI 分析的谱）
- 轮询分析结果中 → 展示进度提示「正在生成谱面…」
- 分析完成 → 自动填充和弦网格，YouTube 音频自动导入第一条 DAW 轨道

### 区域 3：PDF 乐谱视图
- 有上传 PDF → 展示 PDF 页面，可上下滚动
- 无上传 PDF → 展示上传入口卡片「点击上传乐谱 PDF」
- 上传中 → loading 状态

### 区域 4：DAW 面板（底部常驻）

#### 播放控制
- 播放 / 暂停 / 停止 / 回到开头
- 时间显示：当前时间 / 总时长（格式 `0:00 / 3:24`）
- 进度条：可拖拽定位，节拍刻度可视化
- 主音量滑块（0–100%）

#### 轨道列表
- 每条轨道：名称、色块、静音按钮、独奏按钮、音量滑块、声像旋钮
- 静音激活 → 轨道波形变灰
- 独奏激活 → 其他轨道自动静音
- 点击「+ 添加轨道」→ 新建空轨道，命名默认「Track N」
- 点击轨道名 → 可重命名
- 右键 / 长按轨道 → 出现「删除轨道」选项

#### 录音流程
- 点击轨道「录音臂」按钮 → 轨道进入待录状态（红色高亮）
- 按下全局「录音」按钮 → count-in 倒数，然后开始录音
- 录音中 → 实时显示波形
- 按停止 → 波形 clip 存入轨道
- 多次录音 → 多段 clip 叠加在同一轨道时间轴上

#### 未保存离开提示
- 有未提交更改，点击返回/跳转 → 弹出对话框「是否保存？」
- 「保存」→ 保存后跳转
- 「不保存」→ 直接跳转，丢弃更改
- 「取消」→ 留在当前页

## 四、编辑器 `/editor` / `/editor/:id`

### 区域 1：顶栏

#### 项目名称
- 默认名称「未命名项目」
- 双击文字 → 进入内联编辑
- Enter 或点击其他区域 → 确认保存名称
- 空白提交 → 还原上一个有效名称

#### 元数据编辑
- 调式 / 拍号 / BPM → 下拉选择器
- 修改 BPM → 实时更新 DAW 节拍网格

#### 右侧操作
- 「添加段落」按钮 → 下拉菜单（前奏 / 主歌 / 副歌 / 桥段 / 尾奏 / 自定义）
  - 选择类型 → 在网格末尾追加新段落，默认 4 小节
- 视图切换：Lead Sheet ↔ PDF
- 保存按钮（逻辑同播放页）

### 区域 2：和弦网格编辑
- 点击空小节 → 激活输入框，显示光标
- 输入和弦（空格分隔多个，如 `Am7 Dm G`）
- Tab 键 → 移动到下一个小节
- Shift + Tab → 移动到上一个小节
- Enter → 确认输入，退出编辑状态
- Esc → 取消编辑，还原该小节内容
- 点击段落标题右侧「×」→ 删除整个段落（需确认弹窗）

### 区域 3：PDF 导入
- 点击上传 → 选择本地 PDF 文件
- 上传中 → loading
- 上传完成 → 切换到 PDF 视图，展示乐谱
- 已有 PDF，再次上传 → 替换现有文件（需确认）

### 区域 4：DAW 面板
- 逻辑同播放页 DAW 面板
- 新项目默认 1 条空轨道

## 五、演奏中心 `/jam`（Hub 页）

### 区域 1：AI 音色提示输入
- 占位文案：「描述你想要的音色 — 如 'warm fingerpicking tone'…」
- 提交 → 触发 AI 音色生成（当前为 mock，跳转至 `/jam/:id`）

### 区域 2：功能入口卡片（3 列）
- AI 效果踏板 → 点击（暂先不做，入口置灰或跳转占位页）
- AI 乐谱 → 点击跳转搜索页 `/search`
- 更多 AI 工具 → 点击（暂先不做）

### 区域 3：今日推荐音色
- 渐变背景卡片，中心耳机图标
- 风格标签（Blues / Overdrive / Fingerpicking…）
- 点击 → 跳转 `/jam/:id` 并加载该音色（暂先不做，点击无响应）

### 区域 4：我的装备
- 有保存的效果链 → 展示名称 + 简要描述 + 「加载」按钮
  - 点击「加载」→ 跳转 `/jam/:id`，恢复上次效果链配置
- 无保存记录 → 展示空态「尚未保存任何装备」

### 区域 5：装备商店新品（3 列）
- 卡片：Sparkles 图标 + 名称 + 一句话描述
- 点击 →（暂先不做，入口预留）

## 六、即兴演奏页 `/jam/:id`

### 区域 1：会话设置栏

#### 调式选择
- 12 个按钮：A / A# / B / C / C# / D / D# / E / F / F# / G / G#
- 选中状态：高亮背景
- 切换调式 → 实时更新伴奏生成参数

#### BPM 控制
- 数值输入框（可手动输入）
- 范围：40–240
- 左右微调按钮 ± 1
- 下方滑块同步拖拽
- 修改 BPM → 总时长实时重算 → DAW 节拍网格同步更新
- 节拍器开关：开 → 播放时叠加节拍声；关 → 无节拍声

#### 风格选择
- 8 个按钮：Rock / Jazz / Blues / Funk / Lo-fi / Latin / R&B / Electronic
- 单选，切换时高亮更新

### 区域 2：伴奏轨控制

#### 播放控件
- 播放 / 暂停 / 停止 / 回到开头
- 时间轴进度条，可拖拽
- 当前时间 / 总时长

#### 主音量
- 滑块控制（0–100%）
- 实时生效

#### 曲库选择
- 点击「曲库」按钮 → 打开浮层，展示已保存轨道 / 律动列表
- 选中 → 加载对应音频

#### AI 生成伴奏
- 文字提示输入框（如 `chill jazz trio in Dm`）
- 点击「生成」按钮
  - 生成中 → 按钮 loading 状态，提示文案「正在生成…」
  - 生成完成 → 自动加载至伴奏轨
  - 生成失败 → Toast 错误提示 + 保留上一次伴奏

### 区域 3：DAW 多轨录音
- 预置轨道：鼓轨、贝斯轨
- 16 小节默认会话长度
- 逻辑同播放页 DAW 面板

## 七、我的项目 `/projects`

### 区域 1：筛选标签
- All / Scores（学习） / Jam（演奏） / Create（创作）
- 切换标签 → 列表即时过滤，无需刷新

### 区域 2：项目卡片列表（3 列网格）
- 每张卡片元素：
  - 模块图标（BookOpen / Music / Layers / FolderOpen…）
  - 项目名称
  - 可选描述（1 行截断）
  - 最后编辑时间（相对时间：2m ago / 1h ago / 3d ago）
  - 「打开」按钮 + 箭头图标
- 点击「打开」→ 路由跳转规则：
  - `space: learn` + songId → `/play/:songId`
  - `space: jam` → `/jam/:id`
  - `space: editor` → `/editor/:id`
  - 其他 → `/projects`（兜底）
- 排序：按 updatedAt 降序（最近编辑优先）

### 区域 3：空状态
- 当前标签无项目 → 展示空态图标 + 文案「还没有项目，去创建一个吧」
- 提供快捷入口按钮「新建乐谱」「开始演奏」

### 新建项目
- 右上角「+ 新建项目」按钮 → 跳转 `/editor`（空白编辑器）

## 八、AI 助手（Agent Panel）

### 触发方式
- 桌面端：右下角浮动 Bot 图标，点击展开 / 收起
- 移动端：内嵌于页面底部流
- 首页不展示 Agent 浮动按钮
- Escape 键 → 关闭面板

### 欢迎状态（无对话记录时）
- 展示 4 个快捷操作按钮：
  - 「分析乐谱」→ 预填提示词，分析当前页面的和弦结构
  - 「转录音频」→ 引导上传音频文件进行乐谱转录
  - 「建议和弦」→ 基于当前调式/风格推荐和弦进行
  - 「生成伴奏」→ 触发伴奏生成流程

### 对话状态
- 消息列表：用户消息（右对齐）+ 助手消息（左对齐）
- 流式渲染：助手回复逐字出现，显示光标动画
- 支持 Markdown：标题、加粗、列表、代码块
- 底部输入框：发送按钮 + 快捷操作栏（折叠态）
- 流式进行中 → 输入框禁用，不可发送新消息

### 上下文感知
- 当前在播放页 → 系统提示包含「当前歌曲」上下文
- 当前在演奏页 → 系统提示包含「当前调式/风格」上下文
- 切换页面 → 上下文随之更新，历史消息保留

## 附 A：核心用户流程图

### 流程一：搜索 → 生成 → 学习

```
用户输入搜索词
    ↓
展示 YouTube 结果列表
    ↓
用户点击某首歌
    ↓
弹出操作弹窗
    ↓
是否需要乐谱？
  ├── 是 → 选「AI 乐谱 + 伴奏轨」→ 轮询 AI 分析（下载→分析→处理）
  │              ↓ 完成
  │         跳转播放页，和弦网格自动填充，音频导入 DAW 第一轨
  └── 否 → 选「仅生成伴奏轨」→ 跳转播放页，仅音频轨，和弦网格空白
```

### 流程二：空白创作 → 保存 → 恢复

```
点击「新建乐谱」
    ↓
进入空白编辑器 /editor
    ↓
设置元数据（调式/BPM/拍号）+ 添加段落 + 填入和弦
    ↓
在 DAW 面板录音
    ↓
点击「保存」→ 写入 projectStore
    ↓
进入「我的项目」→ 卡片出现
    ↓
点击「打开」→ 恢复编辑器，所有状态还原
```

### 流程三：即兴演奏会话

```
进入演奏中心 /jam
    ↓
选择调式 + BPM + 风格
    ↓
是否有 AI 伴奏需求？
  ├── 有 → 输入提示词 → 点击生成 → 等待 → 自动加载伴奏轨
  └── 无 → 从曲库选择已有伴奏轨
    ↓
按播放 → 伴奏开始，节拍器（可选）同步
    ↓
在 DAW 面板开始录音（录制自己的演奏）
    ↓
结束后保存会话 → 写入 My Projects
```

## 附 B：状态 / 边界情况汇总

| 页面 | 场景 | 处理方式 |
|------|------|----------|
| 搜索结果 | 无搜索结果 | 空态 + 「换个关键词试试」 |

| 编辑器 | 项目名为空提交 | 还原上一次有效名称 |
| 编辑器 | 删除有内容的段落 | 弹确认框 |
| 编辑器 | 未保存直接离开 | 「是否保存？」对话框 |
| 演奏页 | AI 伴奏生成失败 | Toast 报错，保留上一次伴奏 |
| 我的项目 | 无项目 | 空态 + 快捷创建入口 |
| DAW | 录音轨数量上限 | （暂未限制，待确认上限值） |
